# Telegram Mini App UI (Grid Bot)

This repository contains **only** the static UI for the Telegram Mini App.

- Source files: `./webapp/`
- This repo is safe to make **public** (no tokens/keys should be committed).
- The backend logic and secrets live in your local bot project.

## Deploy

GitHub Pages is deployed automatically via GitHub Actions.

After enabling Pages in GitHub:
- Set the bot's `WEBAPP_URL` to the deployed Pages URL.
